# Advent of Code

## What is AOC?
Advent of Code is a advent calender, created by [Eric Wastl](https://twitter.com/ericwastl), of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language you like.

## My AOC
I am writing my AOC solutions in C#, I have completed parts of AOC 2018-2022 but I have a long way to go to get all of my stars.

### Run project
*This project is based on `net6.0` and `C# 10`*

1. Make sure [.NET core](https://dotnet.microsoft.com/en-us/download) is installed.
2. Clone this repo and cd to the cloned directory.
3. Use `dotnet run` to run the current year's project.
     - To run all the the years use `dotnet run all`
     - To run a spesific year use `dotnet run [year]`
